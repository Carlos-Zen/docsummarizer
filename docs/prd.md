# DocSummarizer PRD

## 1. 产品概述

### 产品定位
AI驱动的文档摘要工具，支持PDF/Word/图片/网页一键摘要。

### 核心价值
- 10秒完成文档摘要
- 支持多种格式
- 本地优先，保护隐私
- 多种导出格式

---

## 2. MVP功能列表

### P0 - 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| F01 文件上传 | 支持PDF/Word/TXT/图片 | P0 |
| F02 AI摘要 | 调用AI API生成摘要 | P0 |
| F03 摘要展示 | 结构化展示摘要内容 | P0 |
| F04 复制导出 | 复制Markdown格式 | P0 |

### P1 - 增强功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| F05 批量处理 | 文件夹批量摘要 | P1 |
| F06 历史记录 | 本地存储历史摘要 | P1 |
| F07 模板选择 | 多种摘要模板 | P1 |
| F08 语言选择 | 中英文摘要 | P1 |

---

## 3. 用户流程

```
上传文件 → AI处理 → 展示摘要 → 复制/导出
   (2s)      (10s)     (1s)       (1s)
```

### 核心页面
1. 首页 - 文件上传区
2. 摘要页 - 结果展示
3. 历史页 - 历史记录

---

## 4. 技术架构

```
Frontend (Vue 3)
    ↓
API Gateway
    ↓
AI Service (OpenAI API / 本地模型)
    ↓
Storage (IndexedDB)
```

---

## 5. 非功能需求

| 需求 | 指标 |
|------|------|
| 响应时间 | < 15秒完成摘要 |
| 并发支持 | 单用户，无需并发 |
| 数据安全 | 本地存储，可选云同步 |
| 兼容性 | Chrome 90+ |
